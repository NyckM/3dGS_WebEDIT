Abrir a "cozinha" na nuvem

Clique abaixo para abrir o processador no Google Colab (grátis, precisa de conta Google).

🚀 Abrir o processador
Lá dentro, faça só isto:
1. Menu Ambiente de execução → Alterar tipo → escolha GPU (T4) e salve.
2. Menu Ambiente de execução → Executar tudo (Ctrl+F9).
3. Quando uma célula pedir, envie o arquivo do Passo 1.
4. No final, o resultado baixa sozinho. Só esperar. ☕
